Permutations

This module defines the core Perm type and basic operations on permutations in OrbitAl.jl.

Data Type: Perm

OrbitAl.permutation.Perm — Type
Perm

A permutation represented by a list of integers. The entry at index i indicates the image of i under the permutation.

Use Perm(n) for the identity permutation of size n, or Perm(n, cycles) to construct from cycle notation.

Examples

julia> using OrbitAl

julia> p = Perm([6, 4, 7, 2, 5, 9, 8, 3, 1])
Perm([6, 4, 7, 2, 5, 9, 8, 3, 1])

julia> p == Perm(9, [[1, 6, 9], [2, 4], [3, 7, 8]])
true

julia> id9 = Perm(9)
Perm([1, 2, 3, 4, 5, 6, 7, 8, 9])

julia> p == id9
false

OrbitAl.permutation.transpositions — Function
transpositions(n)

Return the list of adjacent transpositions (j-1 j) for j = 2, …, n. These are the standard Coxeter generators of the symmetric group Sₙ.

Examples

OrbitAl.permutation.permuted — Function
permuted(list, perm)

Return list rearranged by the inverse of perm: permuted(list, perm)[i] == list[i^inv(perm)]. This is the standard Pólya action of a permutation on a list.

Examples

julia> using OrbitAl

julia> permuted([10, 20, 30], Perm([2, 3, 1]))
3-element Vector{Int64}:
