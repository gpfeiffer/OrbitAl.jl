Permutation Groups

This module provides lightweight permutation groups built on top of the orbit algorithms. A group is defined by a list of generators and an identity element; all group-theoretic operations (size, membership, conjugacy classes) are derived from orbit computations.

Types

OrbitAl.permgroup.PermGp — Type
PermGp(gens, one)

A permutation group with generators gens::Vector{Perm} and identity element one.

Examples

julia> using OrbitAl

julia> s = Perm([2,1,3]); t = Perm([1,3,2]);

julia> G = PermGp([s, t], one(s));

julia> sizeOfGroup(G)
6

Size and Membership

OrbitAl.permgroup.sizeOfGroup — Function
sizeOfGroup(group)

Return the order of group using the orbit-stabilizer theorem, without enumerating all elements.

Examples

julia> using OrbitAl

julia> s = Perm([2,1,3]); t = Perm([1,3,2]);

julia> sizeOfGroup(PermGp([s, t], one(s)))
6
source
OrbitAl.permgroup.memberOfGroup — Function
memberOfGroup(group, perm)

Return true if perm belongs to group, using a recursive sifting algorithm.

Examples

julia> using OrbitAl

julia> s = Perm([2,1,3]); t = Perm([1,3,2]);

julia> G = PermGp([s, t], one(s));

julia> memberOfGroup(G, s)
true

julia> memberOfGroup(G, Perm([2,3,4,1]))
false
