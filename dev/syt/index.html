Standard Young Tableaux

This module provides tools for working with integer partitions and standard Young tableaux, built on top of the orbit machinery.

Newton Sums and Differences

Mutually inverse transformations on integer vectors:

OrbitAl.syt.newtonDif — Function
newtonDif(list)

Compute the Newton forward differences of list. Returns a vector whose first element is list[1] and whose remaining elements are the differences between consecutive elements.

Inverse of newtonSum: newtonDif(newtonSum(list)) == list.

Examples

julia> using OrbitAl

julia> newtonDif([1, 3, 6, 10])
4-element Vector{Int64}:
 1
 2
 3
 4
source
OrbitAl.syt.newtonSum — Function
newtonSum(list)

Compute the Newton partial (cumulative) sums of list. The k-th element of the result is list[1] + list[2] + … + list[k].

Inverse of newtonDif: newtonSum(newtonDif(list)) == list.

Examples

julia> using OrbitAl

julia> newtonSum([1, 2, 3, 4])
4-element Vector{Int64}:
  1
  3
  6
 10
source
OrbitAl.syt.newtonDifR — Function
newtonDifR(list)

Right-handed Newton differences: the mirror image of newtonDif applied right-to-left. For a non-increasing sequence (such as a partition), the k-th element of the result is list[k] - list[k+1] (with list[end] as the last element), giving a non-negative vector.

Inverse of newtonSumR: newtonDifR(newtonSumR(list)) == list.

Examples

julia> using OrbitAl

julia> newtonDifR([6, 5, 3])
3-element Vector{Int64}:
 1
 2
 3
source
OrbitAl.syt.newtonSumR — Function
newtonSumR(list)

Right-handed Newton partial sums: the mirror image of newtonSum applied right-to-left. The k-th element of the result is list[k] + list[k+1] + … + list[end].

Inverse of newtonDifR: newtonSumR(newtonDifR(list)) == list.

Examples

julia> using OrbitAl

julia> newtonSumR([1, 2, 3])
3-element Vector{Int64}:
 6
 5
 3

Compositions and Subsets

Bijection between compositions of n and subsets of 1:n-1:

OrbitAl.syt.compositionSubset — Function
compositionSubset(n, set)

Given a subset set ⊆ 1:n-1, return the composition of n whose partial sums form the complement of set in 1:n.

Inverse of subsetComposition.

Examples

julia> using OrbitAl

julia> compositionSubset(9, [4, 5, 7])
6-element Vector{Int64}:
 1
 1
 1
 3
 2
 1
source
OrbitAl.syt.subsetComposition — Function
subsetComposition(n, com)

Given a composition com of n, return the subset of 1:n-1 corresponding to the complement of the partial sums of com in 1:n.

Inverse of compositionSubset.

Examples

julia> using OrbitAl

julia> subsetComposition(9, [1, 1, 1, 3, 2, 1])
3-element Vector{Int64}:
 4
 5
 7

Enumeration

OrbitAl.syt.takeAway — Function
takeAway(set, s)

Remove element s from set and return the resulting set.

Examples

julia> using OrbitAl

julia> takeAway([1, 2, 3], 2)
2-element Vector{Int64}:
 1
 3
source
OrbitAl.syt.subsets — Function
subsets(set)

Compute all subsets of set as the orbit of set under the take-away action. Returns a vector of all subsets (as vectors), starting with set itself.

Examples

julia> using OrbitAl

julia> subsets([1, 2])
4-element Vector{Vector{Int64}}:
 [1, 2]
 [2]
 [1]
 []
source
OrbitAl.syt.partitions — Function
partitions(n)

Enumerate all integer partitions of n as an orbit under a partition-removal action. Returns a vector of non-increasing integer vectors, in reverse lexicographic order (smallest partition first).

Examples

julia> using OrbitAl

julia> partitions(4)
5-element Vector{Vector{Int64}}:
 [1, 1, 1, 1]
 [2, 1, 1]
 [2, 2]
 [3, 1]
 [4]
source

Standard Young Tableaux

OrbitAl.syt.standardYTs — Function
standardYTs(lambda)

Return all standard Young tableaux of shape lambda (a partition given as a non-increasing integer vector). Each tableau is represented as a path in the Young lattice: a vector of (row, col) pairs recording where each number 1, 2, …, n was placed.

Examples

julia> using OrbitAl

julia> length(standardYTs([2, 1]))
2

julia> length(standardYTs([3, 2, 1]))
16
source
OrbitAl.syt.tableau_path — Function
tableau_path(path)

Convert a standard Young tableau path (a vector of (row, col) pairs, as returned by standardYTs) into a row array, where tab[r][c] is the entry placed in row r, column c.

Examples

julia> using OrbitAl

julia> paths = standardYTs([2, 1]);

julia> tableau_path(paths[1])
2-element Vector{Array{Int64}}:
 [1, 2]
 [3]

julia> tableau_path(paths[2])
2-element Vector{Array{Int64}}:
 [1, 3]
 [2]
